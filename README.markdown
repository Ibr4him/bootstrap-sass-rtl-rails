# Bootstrap::Sass::Rtl

RTL version of [bootstrap-sass](https://github.com/thomas-mcdonald/bootstrap-sass).

## Usage

Gemfile:

    gem 'sass-rails', '~> 3.2'
    gem 'bootstrap-sass', '~> 2.3.2.0'
    gem 'bootstrap-sass-rtl', '~> 2.3.2.0'

### CSS

Import in your scss file:

    @import "bootstrap-rtl";
